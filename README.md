# Shelvr

> **Status:** Pre-alpha. Not yet usable. Follow along at your own risk.

Shelvr is a self-hosted ebook library server with a Plex-style architecture: a canonical Windows server holds your library, and clients (Android companion app, third-party OPDS readers, a web admin UI) connect to it.

Part of the [Ink & Iron Apps](https://inknironapps.com) product lineup.
