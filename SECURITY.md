# Security Policy

## Reporting a Vulnerability

If you have discovered a security issue in Shelvr, please report it privately
rather than opening a public GitHub issue.

- **Email:** security@inknironapps.com
- **Response window:** within 5 business days for acknowledgement.

Please include reproduction steps, the affected version (`shelvrd --version`
or the GitHub release tag), and your assessment of impact. We will coordinate
disclosure with you and credit reporters in the release notes once a fix
ships, unless you prefer to remain anonymous.

For non-security questions or general support, see the addresses below.
